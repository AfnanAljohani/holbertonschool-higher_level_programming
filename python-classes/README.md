# Python - Classes and Objects

This project covers Object-Oriented Programming (OOP) in Python:
classes, objects, instances, attributes, private attributes,
properties, getters and setters, and special methods.

## Files
- 0-square.py: Empty Square class
- 1-square.py: Square with size
- 2-square.py: Size validation
- 3-square.py: Area of a square
- 4-square.py: Access and update private attribute via property
- 5-square.py: Printing a square with my_print
- 6-square.py: Coordinates of a square (position)
- 7-singly_linked_list.py: Singly linked list
- 8-square.py: Print Square instance with __str__
- 9-square.py: Compare 2 squares
