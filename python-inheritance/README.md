# Python - Inheritance

This project covers the concept of inheritance in Python.

## Tasks

0. Lookup - Function that returns list of attributes and methods
1. My list - Class inheriting from list with print_sorted()
2. Exact same object - Check exact instance of a class
3. Same class or inherit from - Check inheritance
4. Only sub class of - Check subclass but not direct instance
5. Geometry module - Empty BaseGeometry class
6. Improve Geometry - BaseGeometry with area()
7. Integer validator - BaseGeometry with validation
8. Rectangle - Rectangle inheriting from BaseGeometry
9. Full rectangle - Rectangle with __repr__
10. Square #1 - Square inheriting from Rectangle
11. Square #2 - Square with __repr__
