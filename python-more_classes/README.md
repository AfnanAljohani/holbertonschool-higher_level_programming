# Python - More Classes and Objects

This project extends OOP in Python with the Rectangle class:
class and instance attributes, static methods, class methods,
__str__, __repr__, __del__, and instance counting.

## Files
- 0-rectangle.py: Empty Rectangle class
- 1-rectangle.py: Rectangle with width and height
- 2-rectangle.py: Area and perimeter
- 3-rectangle.py: String representation with __str__
- 4-rectangle.py: __repr__ to recreate instances
- 5-rectangle.py: Instance deletion message
- 6-rectangle.py: Counting instances
- 7-rectangle.py: Customizable print_symbol
- 8-rectangle.py: Compare rectangles by area
- 9-rectangle.py: square class method
