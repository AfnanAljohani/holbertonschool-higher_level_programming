# Python - import & modules

This project covers how to import functions and variables from other
files, create modules, use the built-in `dir()` function, prevent code
from being executed when imported, and handle command line arguments.

## Tasks

- `0-add.py` — Imports `add` from `add_0.py` and prints `1 + 2 = 3`.
- `1-calculation.py` — Imports operations from `calculator_1.py`.
- `2-args.py` — Prints the number of and list of command line arguments.
- `3-infinite_add.py` — Prints the sum of all command line arguments.
- `4-hidden_discovery.py` — Prints all names defined in `hidden_4.pyc`.
- `5-variable_load.py` — Imports the variable `a` from `variable_load_5.py`.
- `100-my_calculator.py` — A simple calculator using `calculator_1.py`.
- `101-easy_print.py` — Prints `#pythoniscool` without `print`.
- `102-magic_calculation.py` — Python version of given bytecode.
- `103-fast_alphabet.py` — Prints the uppercase alphabet.
