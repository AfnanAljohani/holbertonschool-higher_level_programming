# Python - Input/Output

Project covering file reading/writing, JSON serialization and deserialization,
and command line arguments.

## Tasks
- 0-read_file.py: Read a text file
- 1-write_file.py: Write to a file
- 2-append_write.py: Append to a file
- 3-to_json_string.py: Object to JSON string
- 4-from_json_string.py: JSON string to object
- 5-save_to_json_file.py: Save object to JSON file
- 6-load_from_json_file.py: Load object from JSON file
- 7-add_item.py: Load, add, save command line args
- 8-class_to_json.py: Class to JSON dictionary
- 9-student.py: Student class to JSON
- 10-student.py: Student to JSON with filter
- 11-student.py: Student to disk and reload
- 12-pascal_triangle.py: Pascal's Triangle
