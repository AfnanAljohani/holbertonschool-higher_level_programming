The ``4-print_square`` module
=============================

>>> text_indentation = __import__('4-print_square').text_indentation

Normal usage:
    >>> text_indentation("Hello. How are you? Fine: thanks")
    Hello.
    <BLANKLINE>
    How are you?
    <BLANKLINE>
    Fine:
    <BLANKLINE>
    thanks

Invalid type:
    >>> text_indentation(12)
    Traceback (most recent call last):
    TypeError: text must be a string
